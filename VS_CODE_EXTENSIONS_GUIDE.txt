================================================================================
INSTALLATION METHODS
================================================================================

Method 1: Install from VS Code
1. Open VS Code
2. Press Ctrl+Shift+X (open Extensions)
3. Search for extension name
4. Click Install

Method 2: Command Line
# Install multiple extensions at once
code --install-extension ms-vscode.vscode-node-azure-pack
code --install-extension ms-azuretools.vscode-docker
code --install-extension ms-python.python
code --install-extension ms-python.vscode-pylance
code --install-extension humao.rest-client

Method 3: VS Code Settings (Sync across devices)
Settings > Profiles > Create Profile > Include Extensions

================================================================================
CONFIGURATION & BEST PRACTICES
================================================================================

1. Configure Python Path
   File > Preferences > Settings > Search "python.defaultInterpreterPath"
   Set to: path/to/venv/bin/python

2. Configure Black Formatter
   File > Preferences > Settings > Search "black"
   Enable: "Format On Save"
   Set: "Line Length" = 100

3. Configure Docker Path
   File > Preferences > Settings > Search "docker"
   Set correct Docker executable path if needed

4. Configure Azure Account
   - Click Azure icon (left sidebar)
   - Sign in with Microsoft account
   - Select subscription
   - View all resources

5. Azure Activity View
   Open: Command Palette > Azure: Show Output Channel
   View all Azure CLI commands executed

================================================================================
WORKFLOW EXAMPLE: FULL DEVELOPMENT CYCLE
================================================================================

1. Start Development
   - Open folder: File > Open Folder > car_prediction_project
   - Select Python interpreter (venv)
   - Extensions auto-activate

2. Code Development
   - Write Python code
   - PyLint highlights errors in real-time
   - Black formatter auto-formats on save
   - IntelliSense suggests completions

3. Test Locally
   - Run: Right-click > Run Python File
   - Or: Terminal > Run Task
   - View output in Debug Console

4. Docker Build
   - Right-click Dockerfile
   - Select: Build Image
   - View build progress
   - Specify image name: car-predictor:latest

5. Push to ACR
   - Right-click image (Docker view)
   - Select: Push...
   - Choose registry: Azure Container Registry
   - Confirm authentication

6. Deploy to Azure
   - Click Azure icon (sidebar)
   - Right-click Web App (car-price-predictor)
   - Select: Deploy
   - Choose Docker image
   - Wait for deployment

7. Monitor
   - Right-click Web App
   - Select: Streaming Logs
   - Watch real-time logs
   - Test endpoint

8. Git Commit
   - View Git Graph
   - Commit changes
   - Push to GitHub
   - Workflow auto-triggers

================================================================================
TROUBLESHOOTING EXTENSIONS
================================================================================

Extension Not Showing Up:
- Reload: Cmd+Shift+P > Developer: Reload Window
- Reinstall: Uninstall and reinstall

Python Extension Issues:
- Select Interpreter: Cmd+Shift+P > Python: Select Interpreter
- Choose venv/bin/python

Docker Extension Issues:
- Verify Docker running: docker --version
- Check Docker permissions: docker ps

Azure Extension Not Authenticating:
- Sign out: Cmd+Shift+P > Azure: Sign Out
- Sign in again: Cmd+Shift+P > Azure: Sign In

Performance Issues:
- Check extensions using: Cmd+Shift+P > Developer: Show Running Extensions
- Disable unused extensions

RECOMMENDED SETTINGS.JSON
================================================================================

Add these to your VS Code settings.json:

{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "ms-python.black-formatter",
  "python.linting.enabled": true,
  "python.linting.pylintEnabled": true,
  "python.linting.pylintArgs": ["--max-line-length=100"],
  "python.testing.pytestEnabled": true,
  "docker.showDebugingPrompt": false,
  "[python]": {
    "editor.defaultFormatter": "ms-python.black-formatter",
    "editor.formatOnSave": true,
    "editor.codeActionsOnSave": {
      "source.organizeImports": true
    }
  },
  "[json]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "[yaml]": {
    "editor.defaultFormatter": "esbenp.prettier-vscode"
  },
  "files.exclude": {
    "**/__pycache__": true,
    "**/*.pyc": true,
    "**/venv": true,
    "**/.vscode": false
  }
}
